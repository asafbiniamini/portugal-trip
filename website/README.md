- 🏄 תכנית יומית מפורטת עם המלצות לגלישה
- 🍽️ רשימת מסעדות מומלצות עם ציונים גבוהים
- 🍷 יקבים ובתי יין מובילים
- 🌲 מקומות טבע מדהימים
- 🗺️ מפה אינטראקטיבית של פורטוגל
- 📷 אפשרות להעלות תמונות אישיות
- 🔗 קישורים ישירים ל-Google Maps לכל מקום

## איך להשתמש

### פתיחת האתר

1. פתחו את הקובץ `index.html` בדפדפן
2. או השתמשו בשרת מקומי:
   ```bash
   cd website
   python3 -m http.server 8000
   ```
   ואז פתחו: `http://localhost:8000`

### העלאת תמונות

1. בדף הראשי, לחצו על "העלה תמונה שלך" או "העלה תמונה של תם"
2. בחרו תמונה מהמחשב
3. התמונה תוצג אוטומטית באתר

### ניווט

- השתמשו בתפריט העליון כדי לנווט בין הימים
- כל יום מכיל תכנית מפורטת, קישורים ותמונות

## עדכון תכנית

אם תרצו לעדכן את התכנית:

1. ערכו את הקובץ `js/main.js` - שם נמצאים כל הנתונים
2. הרצו את הסקריפט:
   ```bash
   python3 generate_days.py
   ```
3. זה ייצור מחדש את כל דפי הימים

## תכונות נוספות

- **עיצוב רספונסיבי** - האתר נראה טוב גם במובייל
- **ניווט קל** - תפריט ניווט נוח
- **קישורים ישירים** - כל מקום מקושר ל-Google Maps
- **תמונות** - תמונות מהמקומות (מ-Unsplash)

## טיפים

- שמרו את התמונות שלכם בתיקייה `images/` אם תרצו להשתמש בהן
- תוכלו לערוך את ה-CSS בקובץ `css/style.css` לשינוי העיצוב
- כל הקישורים נפתחים בחלון חדש

# איך לפרסם את האתר באינטרנט

יש לך כמה אפשרויות פשוטות לפרסם את האתר כך שהחבר שלך יוכל לראות אותו:

## 🚀 אפשרויות מומלצות (חינם!)

### 1. **Netlify Drop** - הכי פשוט! (מומלץ)

**למה זה הכי טוב:**
- פשוט מאוד - פשוט גרור ושחרר תיקייה
- חינם לחלוטין
- URL מותאם אישית
- עובד מיד

**איך לעשות:**
1. היכנסו ל: https://app.netlify.com/drop
2. גררו את התיקייה `website` לתוך החלון
3. תקבלו URL מיד! (למשל: `https://amazing-trip-123.netlify.app`)
4. שלחו את הקישור לחבר שלכם

**יתרונות:**
- ✅ אין צורך בהרשמה
- ✅ עובד מיד
- ✅ אפשר לעדכן בקלות (פשוט גררו שוב)

---

### 2. **Vercel** - גם מעולה

**איך לעשות:**
1. היכנסו ל: https://vercel.com
2. לחצו על "Add New" → "Project"
3. גררו את התיקייה `website`
4. תקבלו URL מיד

**יתרונות:**
- ✅ חינם
- ✅ מהיר מאוד
- ✅ URL מותאם אישית

---

### 3. **GitHub Pages** - אם יש לך GitHub

**איך לעשות:**
1. צרו חשבון ב-GitHub (אם אין): https://github.com
2. צרו repository חדש
3. העלו את כל הקבצים מתיקיית `website`
4. לכו ל-Settings → Pages
5. בחרו branch: `main` ו-folder: `/ (root)`
6. תקבלו URL: `https://yourusername.github.io/repository-name`

**יתרונות:**
- ✅ חינם
- ✅ קל לעדכן (פשוט push)
- ✅ URL קבוע

---

### 4. **Surge.sh** - פשוט מאוד

**איך לעשות:**
1. התקינו Node.js (אם אין): https://nodejs.org
2. פתחו טרמינל והריצו:
   ```bash
   npm install -g surge
   cd website
   surge
   ```
3. בחרו שם לאתר (למשל: `portugal-trip-2025`)
4. תקבלו URL: `https://portugal-trip-2025.surge.sh`

**יתרונות:**
- ✅ פשוט מאוד
- ✅ חינם
- ✅ מהיר

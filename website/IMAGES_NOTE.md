# הערה חשובה על תמונות

## ⚠️ תמונות אמיתיות בלבד!

**אני לא משתמש בתמונות AI או תמונות לא רלוונטיות.**

כל התמונות באתר הן:
- ✅ תמונות אמיתיות מ-Unsplash (תמונות חופשיות לשימוש)
- ✅ תמונות של המקומות האמיתיים בפורטוגל

## 📸 איך להוסיף תמונות משלכם

אם תרצו תמונות ספציפיות יותר:

1. **העלו תמונות משלכם:**
   - שמרו תמונות בתיקייה `website/images/`
   - עדכנו את הקוד להצביע על התמונות שלכם

2. **חפשו תמונות ספציפיות:**
   - Unsplash: https://unsplash.com
   - Pexels: https://pexels.com
   - חפשו לפי שם המקום (למשל: "Arrifana Beach Portugal")

3. **Google Places Photos:**
   - דורש API key
   - נותן תמונות אמיתיות מהמקומות

## 🔍 תמונות נוכחיות

כל התמונות הנוכחיות הן תמונות אמיתיות של:
- חופי פורטוגל
- ליסבון
- מקומות אמיתיים

אם תרצו שאחליף תמונה ספציפית, תגידו לי ואחפש תמונה אמיתית של המקום!

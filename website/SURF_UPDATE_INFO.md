# עדכון תחזיות גלים - הסבר

## ✅ מה יש עכשיו

הוספתי כפתור **"🔄 עדכן תחזיות גלים"** בדף הגלישה. לחיצה עליו תפתח את תחזיות הגלים המעודכנות באתרים חיצוניים (Magic Seaweed) בחלונות חדשים.

## 🔄 אפשרויות לעדכון אוטומטי

יש כמה אפשרויות לעדכן תחזיות גלים אוטומטית באתר:

### אפשרות 1: Magic Seaweed Widget (הכי פשוט - מומלץ)

**יתרונות:**
- ✅ אין צורך ב-API key
- ✅ חינם
- תחזיות מעודכנות אוטומטית
- ✅ קל להטמעה

**איך לעשות:**
1. היכנסו ל: https://magicseaweed.com
2. חפשו את החוף (למשל: "Arrifana")
3. לחצו על "Embed" או "Widget"
4. העתיקו את קוד ה-HTML
5. הדביקו בדף הגלישה

**דוגמה:**
```html
<iframe src="https://magicseaweed.com/Arrifana-Surf-Forecast/1234/embed/" width="100%" height="400"></iframe>
```

---

### אפשרות 2: Surfline API (דורש API key)

**יתרונות:**
- ✅ תחזיות מדויקות מאוד
- ✅ נתונים מפורטים
- ✅ עדכון אוטומטי

**חסרונות:**
- ❌ דורש הרשמה ו-API key
- ❌ יש הגבלות שימוש

**איך לעשות:**
1. הירשמו ב: https://www.surfline.com
2. בקשו גישה ל-API: https://www.surfline.com/surf-news/surfline-api-access/43246
3. קבלו API key
4. עדכנו את הקוד ב-`js/surf-update.js`

---

### אפשרות 3: Surf Forecast Widget

**איך לעשות:**
1. היכנסו ל: https://www.surf-forecast.com
2. חפשו את החוף
3. לחצו על "Embed" או "Share"
4. העתיקו את קוד ה-HTML

---

### אפשרות 4: עדכון ידני (מה שיש עכשיו)

**מה זה עושה:**
- כפתור שפותח את תחזיות הגלים באתרים חיצוניים
- אתם מעדכנים ידנית את הנתונים באתר

**יתרונות:**
- ✅ פשוט
- ✅ אין צורך ב-API
